명령어

- touch + 파일명 : 파일 생성

- code + 파일명 : 파일 실행

- ctrl + l (clear) : 클리어

- python + 파일명 : 터미널에서 실행

- ctrl + / : 주석처리

- cat + 파일명 : 터미널에서 내용 읽기
